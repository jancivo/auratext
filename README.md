# AuraText - Technical Text Analyzer

A lightweight text processing utility written in Python for string tokenization, structural metric extraction, and simple lexicon analysis.

## Features
- String cleaning and automated token splitting using standard library regex.
- Quantitative syntax calculation (character counts, average word density, sentence distribution).
- Automated Readability Index (ARI) score tracking.
- Lexicon pattern matching for basic sentiment evaluation.

## Requirements
- Python 3.8+
- No external module dependencies.

## Local Usage
Execute the driver script using standard python entrypoint:
```bash
python analyzer.py
```
